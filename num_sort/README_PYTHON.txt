NUMSORT.PY READ ME
===================
This code will take user input of ten(10) numbers and sort the numbers
in descending order. 

SETUP
=====
(running in command prompt)
1. Download and extract the numSort.py file to a location of your choosing

2. Open command prompt and change the directory, with the cd command, to the folder 
location of the file. 
EXAMPLE: C:\Users\%User%\Desktop\Python>

3. Type the command "python" then the file name.
*NOTE* if you do not have python as an enviromental variable this command will not work.
Follow the link to learn how: https://stackoverflow.com/questions/17953124/python-is-not-recognized-as-an-internal-or-external-command
EXAMPLE: >python numSort.py

4. This will run the script. After this follow the directions of the script.
